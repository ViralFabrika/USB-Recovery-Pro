# USB Recovery Pro V6.9

Professional USB File Recovery & Safe Copy Tool for Windows.

## Features

- Automatic USB detection
- Safe Copy Mode
- Category-based file copy
- Progress tracking
- Recovery report generation

## How to use

1. Connect your USB drive.
2. Select the destination folder.
3. Click Start.

## Safety

This tool never writes to the source USB drive.

It only copies visible files.
